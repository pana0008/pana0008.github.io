What did I know about programming before coming to the Netherlands?

I have always been fascinated with programming but for a while I was also really intimated by it and scared to try it, because I thought that I wasn't smart enough for it. But in the summer of 2022, on an Erasmus project with my school we had three programming classes about the basics of HTML and CSS. There I first tried programming and I wasn't scared to learn even more about it anymore.

So in September I decided to start a two-month free course on the basics of Python at SoftUni - a bulgarian educational company that offers all kinds of courses that have to do with the IT field. I finished the course with a certificate at the end of October and I decided that I want to dive in even deeper so I signed up for the next course - Programming Fundamentals with Python. I earned the certificate for this one two, but sadly I couldn't continue to the next one, because school kept me really busy and it was my main priority to graduate successfully.
